# 🏰 랑그릿사 부유도시 계약 최적배치 시뮬레이터
> **Langrisser Floating City Contract Optimizer (LFCO)**

이 프로젝트는 모바일 게임 **'랑그릿사 모바일'**의 부유도시 시스템 중 하나인 **[계약]** 콘텐츠를 위한 고정밀 시뮬레이터 및 자동 최적화 도구입니다. 복잡한 진영 간 시너지와 퍼즐별 고유 그래프 구조를 분석하여 사용자가 보유한 영웅 풀 내에서 최대 보너스 점수를 도출합니다.

---

## 💡 프로젝트 비전
부유도시 계약 퍼즐은 단순한 배치를 넘어 **'제한된 자원(영웅)'**을 **'4개의 서로 다른 요구사항(계약지)'**에 효율적으로 분배해야 하는 조합 최적화 문제입니다. 본 시뮬레이터는 게이머가 수동으로 수백 번의 시행착오를 겪는 대신, 알고리즘을 통해 최적의 해답을 제시하고 직관적인 UI로 이를 시각화합니다.

## ✨ 핵심 기능

### 1. 전방위 영웅 데이터베이스 (230+ Heroes)
- 최신 영웅까지 포함된 방대한 데이터베이스.
- 진영(Faction) 및 클래스(Class) 정보 완전 동기화.
- 영웅별 활성화 클래스(On/Off) 설정 기능 (배치 가능 여부 결정).

### 2. 고정밀 그래프 기반 최적화 엔진
- **Greedy Seed & Iterative Swap**: 탐욕적 알고리즘으로 초기 해를 구한 뒤, 수천 번의 스왑 연산을 통해 점수를 개선하는 알고리즘 적용.
- **다중 계약지 통합 연산**: 4개의 계약지(시청, 산림연맹, 연구원, 형제회)를 하나의 거대한 최적화 풀로 간주하여 연산.

### 3. 직관적인 시각화 및 피드백
- **Dynamic Edge Rendering**: 진영 중첩(시너지) 정도에 따라 선의 굵기와 발광(Glow) 효과가 실시간으로 변화.
- **Batch vs Single View**: 전체 상황을 보는 대시보드 모드와 정밀 배치를 위한 상세 모드 지원.
- **점수 실시간 시뮬레이션**: 영웅을 드래그하여 배치할 때마다 즉각적인 점수 합산 및 시너지 반영.

### 4. 사용자 편의성
- **드래그 앤 드롭**: 영웅 카드를 퍼즐 슬롯에 직접 배치하는 직관적인 UX.
- **필터링 시스템**: 진영별, 클래스별 필터를 통한 효율적인 영웅 관리.

## 🧩 최적화 알고리즘 상세 (Technical)
- **Problem Type**: 본 퍼즐은 NP-Hard 성격을 띠는 그래프 레이아웃 및 레이블링 최적화 문제입니다.
- **Scoring**: 
  - 기본 점수: Node당 1점
  - 시너지 점수: Edge당 중첩 진영 수 (최대 3점)
- **Heuristics**: 연결 선이 많은 노드(High Degree Nodes)에 시너지 잠재력이 높은 영웅(Multi-Faction Heroes)을 우선 배치하는 전략을 사용합니다.

## 🛠 기술 스택
- **Core**: React 19, TypeScript
- **Styling**: Tailwind CSS
- **Visualization**: SVG Dynamic Graph Rendering
- **Deployment**: Gemini App (Web-ready)
