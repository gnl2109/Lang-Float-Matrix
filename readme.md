
# 🏰 랑그릿사 부유도시 계약 최적배치 시뮬레이터

랑그릿사 모바일의 부유도시 '계약' 콘텐츠 점수를 극대화하기 위한 최적화 도구입니다. 복잡한 진영 네트워크를 분석하여 최상의 배치 조합을 찾아줍니다.

## ✨ 주요 기능
- **통합 최적화**: 4개 구역(시청, 산림연맹, 연구원, 형제회) 전체를 고려한 점수 최적화.
- **클래스 관리**: 각 영웅의 실제 전직 상태(클래스 On/Off)를 설정하여 정확한 시뮬레이션 가능.
- **드래그 앤 드롭**: 보드 상에서 영웅을 자유롭게 배치하고 교체 가능.

## 🛠 사용 가이드
1. **사이드바**: 보유한 영웅을 클릭하여 Pool에 추가하세요.
2. **내 영웅 관리**: 각 영웅 카드의 '클래스 On/Off' 버튼을 통해 실제 개방된 클래스를 설정하세요.
3. **자동 최적화**: 상단의 '고정밀 자동 최적화' 버튼을 눌러 점수를 극대화하는 배치를 계산하세요.

## 🧮 점수 산정 방식
- 연결된 슬롯의 영웅끼리 겹치는 진영 수에 따라 점수 부여:
  - 1개: 1점
  - 2개: 2점
  - 3개 이상: 3점
- 각 슬롯은 지정된 클래스(보병, 기병 등)가 활성화된 영웅만 배치 가능합니다.
